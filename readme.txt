作品の詳細
■作品 プラトレ
 ・ゲーム概要
  3Dパズルゲームです。
  イルミネーション(線路)を繋げて、電車を走らせ、人を乗せていき、最後の星を目指すゲームです。
  
  3年生の時の作品です。
  星(マップ)の複合回転に合わせた配置オブジェクトの動きや、、配置する線路の配置の調整、人のランダム配置(詰まないように位置を調整)をしており、
  ファイルを使って、スコアのランキングの表示やセーブがされるようにしました。
  軽量化のために、線路のエフェクトを正面(線路を配置するマップ面)と、プレイキャラがいる面でプレイキャラとの距離が近ければ発生させています。
  二画面表示のためにビューポート設定もしています。
