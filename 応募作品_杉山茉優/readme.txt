作品の詳細
■作品 プラトレ
 ・ゲーム概要
 3Dパズルゲームです。
  イルミネーション(線路)を繋げて、電車を走らせ、人を乗せていき、最後の星を目指すゲームです。
  
  2年生の時の作品です。
  星(マップ)の複合回転に合わせたオブジェクトの動きや、、線路の配置位置の調整、人のランダム配置(詰まないように位置を調整)をしており、
  外部ファイルにデータを出力して、ランキング用のスコアの保持や、セーブデータを残すようにしています。
  軽量化のために 線路のエフェクトを正面(線路を配置するマップ面)または、プレイキャラがいる面でプレイキャラから一定距離以内に線路がある場合、発生させています。
  二画面表示はビューポートの設定で実現しています。
  
  
※作品についてvisualstudioが入っていないPCでも確認済みですが、
  VisualStudioでコンパイルして動作させるにはDirectX SDKが必要になります。
